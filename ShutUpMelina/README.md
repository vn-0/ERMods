This mod currently has three variants. More are planned.
- "FirstCutsceneAndDialogue" 	- Skips only the introductory cutscene and dialogue without any further changes.
- "FirstCutsceneOnly"		- Skips only the introductory cutscene. Melina's dialogue is unchanged.
- "FirstDialogueOnly"		- Skips only Melina's dialogue after the introductory cutscene, which itself is unchanged.
